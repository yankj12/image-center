# image-center

用来管理应用中涉及到的图片

## 待实现功能

- 上传图片
- 异步上传图片
- 将图片根据分类存储到硬盘固定位置
- 使用uuid值来命名存储到硬盘的图片，将图片名称和uuid值的映射关系存表
- 计算图片md5值
- 相同md5值的图片不重复保存，只在表中增加记录
- 支持给图片打标签
- 提供图床功能
- 支持根据条件检索图片（名称、分类、标签等）
